# 04 类 Claude Code 能力总结与优先级

## 典型能力概览
- 终端交互式对话
- 代码搜索与语义检索
- 文件读写与批量修改
- 运行命令并解析输出
- 自动生成补丁并预览变更
- 上下文管理与会话记忆
- 工具调用安全约束
- 任务分解与计划执行

## 结合 LoongClaw 的能力对齐
- CLI 为第一入口，提供 REPL 与指令模式
- 工具系统统一封装文件与命令能力
- 以 session 为核心管理上下文
- 输出结构化日志提升可观察性

## 优先级建议
1. CLI 交互稳定性与工具输出可观测
2. 文件系统完整能力（读写/创建/删除/搜索）
3. 命令执行与错误回传（白名单）
4. 语义搜索与多文件修改
5. 记忆系统增强与可导出
6. 工具链扩展与插件化
